# Thema: Reverse Shells Übungen
## Übung netcat chat
### Angabe:  
Verwende netcat um einen bidirektionalen Chat zwischen 2 unterschiedlichen Rechnern aufzubauen.

### Informationen
Ein bidirektionaler Chat bedeutet, dass beide Seiten gleichzeitig senden und empfangen können.

Im Kontext deiner Netcat-Übung heißt das konkret:   
Rechner A kann Nachrichten an Rechner B schicken   
Rechner B kann gleichzeitig Nachrichten an Rechner A schicken   
Es ist also keine Einbahn-Kommunikation, sondern ein Dialog in beide Richtungen   

### Am Rechner A netcad im Listening Modus starten
Befehl:
```
nc -l -p 1234
```
1234 ist der Port, der verwendet wird.   

### Am Rechner B mit Rechner A verbinden
Befehl:   
```
nc 192.168.17.51 1234
```   
192.168.17.51 ist die IP-Adresse des Rechners A, mit dem sich der Rechner B verbinden will.    
1234 ist der verwendete Port.
 
### Chatten   
Alles, was man auf Rechner A eingibt → erscheint auf Rechner B   
Alles, was man auf Rechner B eingibt → erscheint auf Rechner A
